# Dartpedia

Dartpedia is an interactive command-line (CLI) application built in Dart. It is designed as a learning project to explore and demonstrate the essential features of Dart development, including variables, functions, classes, asynchronous programming, testing, and more.

This project follows the official Dart tutorial, guiding you through the process of building a CLI app from scratch. The goal is to provide a hands-on experience for anyone looking to learn Dart by example.

Whether you're new to Dart or looking to reinforce your skills, Dartpedia offers a practical way to understand how Dart works in real-world applications.
